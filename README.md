 ### Let & Const
    Instead of using 'var' , its highly incourage to use 'let' and 'const'.

    let = Variable values
    const = Constat values

    ```js
    let namaku = 'Adli';
    const phi = 3.14;
    ```

## Function 

    Fungsi biasa : 
    ```js
    function namaku(nama){
        console.log(nama);
    }
    namaku('Adli');
    ```
    Fungsi arrow : 
    ```js
    const namalu = (nama) => {
        console.log(nama);
    }
    namalu('WHatever');
    ```
    short version 
    ```js
    const namalu = nama => nama;
    console.log(namalu('Adli'));
    ```
